# 👗 FitAR – AR Virtual Try-On Web Application

## 📌 Description
FitAR is an augmented reality-based web application that allows users to virtually try on clothing in real-time. The system enhances online shopping experience by enabling users to visualize how clothes fit on their body before making a purchase.

## 🛠️ Technologies Used
- Unity3D  
- MediaPipe  
- OpenCV  
- Web-based integration
- Full-stack development

## 🎯 Key Features
- Real-time body tracking using camera  
- Virtual clothing overlay on user body  
- Interactive and user-friendly interface  
- Improves user confidence in purchase decisions
